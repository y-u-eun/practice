# linux command
# 각 명령어 별 뒤에 옵션을 붙일 수 있음(생략하고 사용 가능)

ls : 디렉토리 목록 확인
( -al 옵션으로 디렉토리 목록에서 파일, 디렉토리에 대해 권한 등 자세하게 알 수 있음 )
--help : 명령어의 사용 방법을 알려 줌( [명령어] --help )
rm : 파일 또는 디렉토리 지우기
( rm [옵션] [삭제할 파일 또는 디렉토리]   ; 옵션 -rf는 현재 디렉토리의 모든 파일과 디렉토리 삭제 )
cp : 파일 또는 디렉토리 복사( cp [복사할 파일 또는 디렉토리] [복사될 위치] )
mv : 파일 또는 디렉토리 이동( mv [이동할 파일 또는 디렉토리] [이동될 위치] )
cat : 파일의 내용을 확인할 수 있음
mkdir : 새로운 디렉토리 생성( mkdir [디렉토리 이름] )
chmod : 권한 변경( chmod ___ [파일이름] )
cd : 디렉토리 이동( cd [이동할 경로]    ; cd ..은 상위 폴더(디렉토리)로 이동 ; cd .은 현재 위치한 폴더(디렉토리)로 이동 )
ps : 사용자와 관련된 프로세스 출력
uname : 특정 시스템 정보를 출력
